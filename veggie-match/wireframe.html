4. 计分与连击系统

基础得分

消除类型基础分说明
3连线 (match_3)50最基本的消除
4连线 (match_4)120生成条纹特殊蔬菜
5连线 (match_5)300生成彩虹蔬菜
L/T型 (match_L_T)200两条3连线交叉
条纹激活+150消除整行/列额外加分
彩虹激活+500全屏同色消除额外加分

连锁倍率 (Cascade Multiplier)

公式：score = base_points x (1 + 0.5 x (cascade_level - 1))
第1次消除: 1.0x | 第2次: 1.5x | 第3次: 2.0x | 第4次: 2.5x | ... | 最高: 5.0x

6. 音频清单

音频风格： cartoon (卡通风格，与视觉风格一致)

BGM： 无（用户不需要背景音乐）

音效模式： 全部使用 synthesized（Web Audio API / sfxr 合成）

音效 ID 触发事件 模式 听感描述

8. 核心机制设计依据

Match-3 核心循环

输入 → 匹配 → 消除 → 下落 → 级联 → 得分/目标
玩家交换相邻蔬菜 → 检测3+连线 → 匹配的蔬菜消失（视觉+音效） → 上方蔬菜下落填充 → 检查新匹配（级联/cascade） → 计算得分和目标进度

难度设计多维度递进

  • 棋盘大小：6x6 → 7x7 → 8x8 → 8x8 → 9x9（更大=更多蔬菜，但也更难找到最优匹配）
  • 蔬菜种类：4 → 5 → 5 → 6 → 6（种类越多，随机匹配概率越低）
  • 步数限制：25 → 22 → 20 → 18 → 15（容错空间递减）
  • 目标数量：2种各10个 → 3种10-15个 → ... → 6种各15-25个
